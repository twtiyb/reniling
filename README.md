spring,springmvc(struts),mybatis  app三层
memoched,tomcat,apache  容器及中间件。apache做转发，tomcat端口不同多开。
oracle 数据库。
bootstrap+一系列的前端插件  当前要做的就是将前端插件与后台数据做交互结合。使业务上的交互不用再去写js。只需要写后端即可。
